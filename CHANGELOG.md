# 📝 Changelog

<details open>
<summary><b>🇨🇳 v0.50 更新说明 (Chinese)</b></summary>

### 🎉 新增 (New Features)
- **全新极客风个人博客主页**：将原有简单的导航页彻底重构为一个完整的、具有赛博朋克黑客帝国风格的个人博客与项目展示橱窗。
- **打字机动态交互**：主页新增 `<SYSTEM_INIT />` 终端打字机启动动画。
- **全新大模型聊天体验模块**：新增名为 **Synapse AI** 的前端大语言模型聊天模拟页面（`/chat`）。该模块拥有沉浸式对话流、模拟神经元计算动画以及赛博朋克极光背景。

### 🧹 优化 (Optimizations)
- **统一导航系统**：对 `Cyber Match` 记忆翻牌与 `Neon Tetris` 俄罗斯方块两款旧游戏增加了与 `Synapse AI` 视觉一致的“返回主站”导航。
- **性能及历史清理**：按照部署规范升级项目版本至 `0.50`，清理旧版无用代码，压缩 Git 历史为单一记录，进一步减小项目体积。
</details>

<details>
<summary><b>🇬🇧 v0.50 Release Notes (English)</b></summary>

### 🎉 New Features
- **All-New Geek Portfolio Homepage**: Completely redesigned the old simple navigation page into a full-fledged, cyberpunk/Matrix-style personal blog and project showcase.
- **Typewriter Dynamic Interactions**: Added a `<SYSTEM_INIT />` terminal typewriter boot animation on the homepage.
- **Brand New LLM Chat Experience Module**: Introduced **Synapse AI**, a frontend Large Language Model chat simulation page (`/chat`). This module features an immersive conversation flow, simulated neural computation animations, and a cyberpunk aurora background.

### 🧹 Optimizations
- **Unified Navigation System**: Added a "Back to Home" navigation to the existing `Cyber Match` and `Neon Tetris` mini-games to match the visual consistency of `Synapse AI`.
- **Performance & History Cleanup**: Upgraded project version to `0.50` per deployment guidelines, cleaned up obsolete code, compressed Git history into a single record, further reducing project size.
