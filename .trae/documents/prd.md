## 1. 产品概述
在当前的 Next.js 项目中，将原有的简单游戏导航主页（`/`）重构为一个**个人博客风格（类似 GitHub 个人主页/网络博客）的主页**。同时，在合集中新增一个**大语言模型（AI LLM）聊天体验页面**（暂命名为 "Synapse AI"），并提供统一的导航体系。

### 核心目标：
- 打造一个展示个人技能、项目介绍的极客风/赛博朋克风的博客主页。
- 整合现有两款小游戏（Cyber Match, Neon Tetris）并在博客主页展示入口。
- 新增一个能够提供前端大模型聊天交互界面体验的新功能页面。
- 保证所有子页面（小游戏、AI 聊天）均能方便地跳转回主页。

## 2. 核心功能与模块

### 2.1 个人博客主页 (`/`)
- **头部信息 (Hero Section)**：个人介绍、社交链接徽章（如 GitHub, Twitter）、打字机效果的欢迎语。
- **项目橱窗 (Projects/Games Gallery)**：以网格卡片的形式展示已有的小游戏以及新的 AI 聊天页面。每张卡片包含：项目名称、描述（中英双语）、技术栈标签、"Try it out" 按钮。
- **多语言切换**：保留右上角的中英文切换功能，切换后影响全站文本。

### 2.2 AI 大模型聊天体验页 (`/chat`)
- **页面名称**：Synapse AI
- **UI 布局**：现代化的聊天窗口设计，左侧边栏（可选或折叠），右侧主聊天区（对话气泡流、底端输入框）。
- **交互功能**：
  - 用户输入文字并回车/点击发送，消息上屏。
  - AI 显示“思考中”动画（Typing effect/Loading bubbles），随后给出预设或模拟的回复（由于没有真实后端 API，可以通过前端模拟回复或接入免费公共 API 模拟交互体验）。
  - 支持多行输入、代码块高亮显示。
- **导航功能**：左上角明显的“返回主页”按钮。

### 2.3 现有游戏子页面调整 (`/game`, `/tetris`)
- 确保页面左上角或右上角拥有统一、美观的“&lt; 返回主页 (Back to Home)”导航链接。

## 3. 用户界面设计

### 3.1 设计风格
- 继续保持项目现有的**赛博朋克 / 暗黑极客风格**。
- 主页采用 Markdown 风格排版或卡片式瀑布流，背景使用深黑（`bg-black`）搭配网格纹理或极光渐变背景。
- 文字使用无衬线字体及等宽字体组合（Sans-serif + Monospace）。

### 3.2 交互效果
- **卡片悬停 (Hover)**：光标悬停在项目卡片上时，卡片边缘发光（Glow）、略微上浮。
- **打字机动画**：AI 回复和主页标语具有逐字显示的动画。
